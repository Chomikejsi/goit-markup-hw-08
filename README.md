# WebStudio - responsive agency website

Live demo: https://chomikejsi.github.io/webstudio-agency/

WebStudio is a responsive agency landing page prepared as a portfolio project. It focuses on clean structure, responsive layout, polished UI states, modal interactions and a production-ready GitHub Pages demo.

## What this project shows

- semantic HTML structure,
- responsive layout for mobile, tablet, and desktop,
- retina-ready images with `picture` and `srcset`,
- SVG icon sprites,
- modal and mobile menu interactions,
- portfolio cards with hover overlays,
- contact and newsletter form markup.

## Tech stack

- HTML5
- CSS3 / Sass output
- JavaScript
- Modern Normalize
- Google Fonts
- GitHub Pages

## Pages

- `index.html` - agency landing page
- `portfolio.html` - project grid

## Why it matters for my portfolio

This repository demonstrates frontend fundamentals: layout, semantics, responsive assets, UI states, and the ability to improve older code so it is clearer for both users and reviewers.
